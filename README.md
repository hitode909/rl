# rl.rb

Print the term's balance in QWERTY keyboard.

```
% ruby rl.rb 2015
left: 3, right: 1
75% left
% ruby rl.rb 1999
left: 1, right: 3
75% right
% ruby rl.rb github
left: 3, right: 3
same
% ruby rl.rb "pop music"
left: 2, right: 6
75% right
% ruby rl.rb milk
left: 0, right: 4
100% right
% ruby rl.rb ninjinkun
left: 0, right: 9
100% right
```
